# O que é Null Safety e como utilizá-lo?

Em julho de 2020, a segurança nula sonora foi introduzida na linguagem de programação Dart. Como essa linguagem potencializa o Flutter SDK , o suporte para segurança nula foi uma das melhorias previstas no Flutter 2.

O lançamento da nova versão do Flutter em 3 de março de 2021, finalmente tornou possível não apenas escrever um novo código usando o recurso de segurança nula, mas também migrar os aplicativos Flutter existentes para segurança nula.

# Como funciona o Null Safety

Volttando u pouco na história. O ponteiro nulo ou referência nula foi inventado por Tom Hoare em 1965. Mais tarde, em 2009, ele diria que foi um erro de um bilhão de dólares durante seu discurso em uma conferência de desenvolvimento de software.

Tom Hoare estava projetando o primeiro sistema de tipo abrangente para programação orientada a objetos. Para ter certeza de que o uso de todas as referências é seguro, ele decidiu implementar a referência de ponteiro nulo como a solução mais fácil. Como resultado, o erro nulo nasceu.

A Segurança nula (Null Safetty) elimina bugs causados ??por ponteiros nulos. Isso torna os tipos em seu código Dart não anuláveis ??por padrão. Isso significa que as variáveis ??não podem conter nulos, a menos que você diga que podem.

# Razões para migrar um aplicativo Flutter para Null Safety

Existem muitos motivos para migrar aplicativos Flutter para segurança nula. Esse recurso é uma alteração importante e os aplicativos escritos anteriormente não compilarão com o verificador de nulos, o que pode levar alguns desenvolvedores preguiçosos a não ativá-lo.

Você deve saber que, ao fazer a refatoração para segurança nula, pode confiar completamente no compilador. Isso torna o processo bastante simples. Acredita-se que migrar seu código para segurança nula é mais obrigatório do que opcional. Os esforços que você dedicar a isso irão lhe poupar muitas horas de trabalho posteriormente.

Null Safety é um ótimo recurso que permite ao Dart acompanhar outras linguagens como Kotlin e Typescript. Isso reafirma que o Dart é uma linguagem voltada para a felicidade do desenvolvedor. Já era hora de a equipe de desenvolvimento do Flutter nos fornecer esse recurso e ele apenas tornará o Flutter SDK ainda melhor para escrever aplicativos que rodam literalmente em qualquer lugar.
